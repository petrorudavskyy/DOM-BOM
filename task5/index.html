PreventDefault()

Prevent default: Якщо цей метод викликається, дія за замовчуванням події не спрацьовуватиме. Наприклад при кліку на ссилці, вона не буде переходити на посилання

StopPropagation()

Функція event.stopPropagation() дозволяє виконати подію на поточному елементі, але блокує виконання функції на всіх батьківських елементах.Метод stopPropagation (), зупиняє спливання (bubbling) будь-якої події до батьківських елементів.

Для прикладу подивіться на два блоки. Перший з них(лівий) використовує функцію preventDefault().Для правого виконується функція із застосовується event.stopPropagation(). Якщо клікнути по лівому блоці, то функція виконається для поточно елемента(на котрому клікнули) і ще по одному разу для кожного батьківського. У випадку з правим блоком, то функція виконається тільки один раз - для поточного елемента.

Відмінності - перша блокує стандартну дію браузера, друга - виконання події для батьківських елементів.
